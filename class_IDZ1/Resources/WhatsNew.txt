Class IDZ1 
Version 1.2.0.5
Добавлены новые способы сохранения save_IDZ.
Добавлен метод "дозаписи" в файлы кадров, и метод записи шапки в файл.

Version 1.3.0.1
В методах сохранения параметры усиления КУУО (amp1 & amp2) записываются в обычном(обратном) порядке байт(для совместимости файлов idz1 с lookm).
Расширено значение высоты кадра: если высота больше 255, то в байт 3 записывается ноль(что является признаком увеличенной высоты, 
а в байты 41-42 (первый резервный байт) записывается значение высоты.
Удалены все stream.Close() и stream.Dispose() в методах сохранения и чтения с использованием Using (new stream).
Добавлены методы записи и чтения "фрагментов" в кадре. Фрагменты записаны в конце файла, формат записи: UInt16 - количество фрагментов, 
затем идут блоки фрагментов по 4 UInt16, задающие границы фрагментов.
Удалены неиспользуемые ссылки на библиотеки.

Version 1.3.1.1
Добавлен класс для работы с файлами формата TIFF, а также класс формы для отображения заголовочной информации файла TIFF.

Version 1.3.1.2
Исправлена ошибка чтения коэффициентов усиления из файлов idz. В формате idz коэффичиенты усиления записаны в обратном порядке байт, поэтому для нормального числа необходимо разворачивать 2 байта

Version 1.3.2.1
Добавлена функция чтения строк перескана в файлах idz.
